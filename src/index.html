CarDeal

My Role: Designer and developer

Time to complete: 2 weeks

Sources: Smashicons from Flaticon - Icon design

Project Description: CarDeal is a made-up company that offers to buy your car for cash, with the option to do a trade-in to purchase a vehicle CarDeal owns. One of the main goals was to design a website that is more customized and stands out from other competitor websites, while still providing customers with the information they need.
I was given specific instructions for what content to include in each section, but I chose the colors, fonts, and layout to implement. I also designed a favicon for the website so it has more presence in browser tabs. The site is a one-page website that is responsive and displays pleasing looks on desktop, tablet, and mobile phone screens.
I believe one of the strongest features of this web project is the header/hero section. It is extremely important to have the first thing users see be visually pleasing and quickly summarize what the website is about, and the design I have creates a good first impression while making it clear that this website buys cars.

Webby

My Role: Customizer

Time to complete: 2 weeks

Project Description: Webby is a pretend website for a web designer. The main goal of this project was to take a website template and customize it to make it more my own by using SASS to customize Bootstrap's default styles.
The one-page starter project was created from a YouTube tutorial, then I chose custom colors, images, graphics, icons, designed a logo, added real text, and added a couple new sections to the website to make it more complete. The site is responsive, and displays correctly on all screen widths, from mobile to desktop.
One of the struggles I experienced with this project was overriding the default styles created by Bootstrap. After some manipulating of code and experimenting, I was able to overcome this and create a system that allowed me to easily customize the website.

About Me Website

My Role: Designer and Developer

Time to complete: 1 month

Project Description: This informational website is part of a project proposed by one of my college instructors. The main purpose of this website is to combine the skills I learned in my Web 2620 class and gain more experience with coding websites.
Part of this project was to create a mockup using Figma before coding the website. I designed this website from scratch, with only the guidelines my professor gave us, such as to have 5-6 pages and use Bootstrap to help code the website.
One frustration that I had while working on this project was not knowing how to change the Bootstrap code to look less like Bootstrap. Although I did not yet understand the full possibilites of customization, I managed to change the primary color that affected my interactive elements.

RunForLife

My Role: Designer and Developer

Time to complete: 2 weeks

Project Description: RunForLife is a website that provides runners all over the world with a social hub to improve everyone's running experiences. The main functionalities are finding places to run, buy shoes, check their BMI, and read and post stories and tips about running.
The guidelines for this project were to include 5 main pages and incorporate some Javascript into the website, which I fulfilled by creating the BMI calculator to take user inputs and output the calculation result. For the destinations page, I went out and took my own photos of places to go running, as well as edited them in Photoshop.
A struggle I had was creating a popup screen for the destinations with more info about them. However, using Javascript, I was able to create the desired popup that could be closed out with an X button.

Electrol

Role: Designer and Developer

Time to complete: ?

Project Description: Electrol is a fictitious company that I imagined for my final website project in Web 1400. Electrol is an electric vehicle company that sells electric cars, bikes, and even boats. My goal for the website was to create something non-traditional that stands out from other websites.
The professor had some requirements such as creating a fake company to design a website for, along with having 5 pages on the website. I created the company's logo and all the content in the webpage, as well as a mockup of my design for the website.
One of the first things you notice about this website is that the navigation is on the left side of the screen, instead of the usual top. The goal is to create a different look in the website that will cause people to do a double-take, and hopefully stay on the page longer. It also allows the users to access any part of the navigation at any time, with a different style than just having the navbar fixed at the top.

Complex Night Photo

Role: Photographer and Editor

Time to complete: 2 days

Project Description: I took this photo for an assignment in my photography class. The goal was to photograph a pleasing photo while involving some creative light elements.
We were required to take the photo at night, and experiment with longer shutter speeds to capture more light in the darker environment. This photo was the result of seeking out a location with moving lights to create the streaking light effect in the darkness.
Something that works well in this photo is not just the streaks of light, but that there are different colors of streaks to give variety to the photo. When strategically placed on a website, it helps catch the user's eye and guides them to the next thing they should look at.
