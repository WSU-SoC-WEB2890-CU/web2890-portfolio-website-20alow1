Webby

Role: Customizer

Time to Complete: 2 weeks

Sources: Code adapted from this YouTube Course

Full Website (opens in new tab)

Description

Webby is a pretend website for a web designer. The main goal of this project was to take a website template and customize it to make it more my own by using SASS to customize Bootstrap's default styles.

The one-page starter project was created from a YouTube tutorial, then I chose custom colors, images, graphics, icons, designed a logo, added real text, and added a couple new sections to the website to make it more complete. The site is responsive, and displays correctly on all screen widths, from mobile to desktop.

One of the struggles I experienced with this project was overriding the default styles created by Bootstrap. After some manipulating of code and experimenting, I was able to overcome this and create a system that allowed me to easily customize the website.
